- **Glitch logo** — toggles between `D.O.A.` and `DEAD ON ARRIVAL`
- **24H Volatility Map** + **Fate Breakdown** (Pie left, Candle right)
- **Live Slaughter Feed** + Top MEV Predators

- PWA ready (installable)
